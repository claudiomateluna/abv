Permite visualizar productos Movistar bajo sus distintas modalidad de venta, como también crear productos propios del franquiciado para las transacciones **"OTRAS VENTAS"**.

Podremos visualizar, entonces, todos los productos de movistar disponibles y de acuerdo la lista de precios que mantiene Movistar en el POS los que podremos buscar mediante el código siscel que tengan.

<p class="centrado"><img src="{{site.baseurl}}/docs/pos/img/productos/1.png"></p>

También podremos, mediante esta misma instancia, agregar productos propios o similares para ser vendidos mediante el POS

<p class="centrado"><img src="{{site.baseurl}}/docs/pos/img/productos/2.png"></p>

Para agregar un producto propio es necesario especificar: 

Un **código de barras** que le dara un número identificador único<br>
Una **Descripción corta** idealmente el nombre del producto que se incorpora<br>
Una **Descripción larga**  que describa detalladamente el producto<br>
Un **Precio de Producto Propio** siendo este el valor que tendra el producto que se incorpora.<br>
y la opción **Pide Precio en Venta** que permite crear productos propios que soliciten el precio al momento de realizar la venta.

Finalmente presionaremos el botón **Grabar** para almacenar el producto en el sistema.
